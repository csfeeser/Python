### Module Demonstration

```python
# BEST PRACTICE- import your modules FIRST at the top of your script!
import random

# random.randint(x,y) generate a random number between x and y
random_num= random.randint(1,100)
print(random_num)

# random.choice(z)    randomly pick an object from z
letters= ["a","b","c","d"]
print(random.choice(letters))
```
